### 方法B：Mac预览截图（10分钟，最安全）

**步骤：**
1. 找到PDF文件：`02 进阶篇 YTC卷二：市场和市场分析.pdf`
2. 右键 → 打开方式 → 预览（或双击）
3. 在预览应用中打开
4. 滚动到第23-40页（HH+HL部分）
5. **快捷键截图：** `Command + Shift + 4`
   - 截取整个页面
6. 保存到：`images/diagrams/hh-hl-basic.png`
7. 继续滚动，重复步骤5-6

**重命名快捷方式：**
1. 截图后桌面会有"快照"文件
2. 重命名为所需名称
3. 拖拽到 `images/diagrams/` 文件夹

## 💡 专业技巧

### 截图技巧
```
✅ 使用PDF预览的缩放功能放大再截图
✅ 确保包含完整图表（不要截到一半）
✅ 保存为PNG格式（质量最佳）
✅ 文件名使用英文（兼容性好）
```

### 命名规范
```
格式：主题-类型.png
示例：hh-hl-basic.png

避免：
❌ 中文文件名
❌ 特殊字符
❌ 空格
❌ 过长的文件名
```
