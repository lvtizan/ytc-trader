# YTC实战案例标准模板

## 📋 每个案例必须包含的元素

### 1. 基本信息
- 股票代码和名称
- 所属板块
- 时间周期
- 策略类型(PB/CPB/STS)

### 2. 真实K线图 ⭐⭐⭐ (必须)
- **SVG矢量K线图** - 清晰展示完整走势
- **标注关键点位**:
  - 高点/低点价格
  - 回调幅度(%)
  - 入场点/止损点/目标点
  - 持仓时间
  - 实际涨跌幅

### 3. 图表标注
- 趋势线
- 支撑/阻力位
- 回调比例线(38.2%, 50%, 61.8%)
- 入场/出场/止损标记
- 成交量变化

### 4. 文字说明
- 走势描述(简洁)
- 形态特征
- 操作要点
- 风险回报比

## 🎯 K线图制作要求

### 技术规格
```svg
<svg viewBox="0 0 800 450" style="width: 100%; height: auto;">
  <!-- 必须包含:
       1. 网格背景
       2. Y轴价格标签
       3. X轴时间标签
       4. 关键点位标注
       5. 成交量柱状图(底部)
       6. 入场/止损/目标箭头
       7. 回调幅度标注
  -->
</svg>
```

### 视觉要求
- **配色**:
  - 阳线: #27ae60 (绿色)
  - 阴线: #e74c3c (红色)
  - 背景: 半透明暗色
  - 标注线: 清晰可见

- **字体大小**:
  - 价格标签: 12px
  - 标注文字: 11-14px
  - 图表标题: 16-18px

## 🎨 推荐的图表元素

### 入场点标注
```
🔵 蓝色圆圈 + 箭头 + "入场"文字
```

### 止损点标注
```
🔴 红色虚线 + "止损"文字
```

### 目标点标注
```
🟢 绿色虚线 + "目标"文字
```

### 回调幅度
```
黄色双向箭头 + 百分比
```
