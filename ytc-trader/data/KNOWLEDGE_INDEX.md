## 阶段一：市场分析与趋势识别（第1-4周）
**教材**：02 进阶篇 YTC卷二：市场和市场分析.pdf

### 知识点清单

#### 1.1 上升趋势的完整结构
- **知识点1**：摆动高低点定义
  - PDF章节：第3章 - 市场结构
  - 页码参考：p.45-60
  - 关键概念：
    - 摆动高点 (Swing High)
    - 摆动低点 (Swing Low)
    - HH (Higher Higher) 更高的高点
    - HL (Higher Low) 更高的低点

- **知识点2**：上升趋势识别
  - PDF章节：第3章 - 趋势识别
  - 关键要点：
    - 至少两个连续的HH和HL
    - 价格在主要移动平均线上方
    - 回调不破前低

- **知识点3**：市场结构的层次
  - PDF章节：第4章 - 多时间周期分析
  - 关键要点：
    - 主要结构（周线）
    - 中级结构（日线）
    - 小型结构（4小时）

#### 1.2 回调的分类与识别
- **知识点4**：回调的类型
  - PDF章节：第5章 - 回调分析
  - 关键要点：
    - 浅回调 (<33%)
    - 中等回调 (33-50%)
    - 深度回调 (50-66%)

- **知识点5**：斐波那契回调位
  - PDF章节：第5章 - 斐波那契
  - 关键比率：
    - 38.2%
    - 50%
    - 61.8%

- **知识点6**：回调时的成交量特征
  - PDF章节：第6章 - 成交量分析
  - 关键要点：
    - 健康回调：成交量萎缩
    - 不健康回调：成交量放大

#### 1.3 支撑与阻力
- **知识点7**：结构性支撑阻力
  - PDF章节：第7章 - 支撑阻力
  - 关键要点：
    - 前期高点转为支撑
    - 前期低点转为阻力
    - 整数关口

- **知识点8**：动态支撑阻力
  - PDF章节：第7章 - 移动平均线
  - 关键均线：
    - 20日均线
    - 50日均线
    - 200日均线

#### 1.4 选股系统
- **知识点9**：趋势过滤器
  - PDF章节：第8章 - 市场环境分析
  - 检查清单：
    - 周线趋势向上
    - 日线趋势向上
    - 价格在20日均线上方

- **知识点10**：选股5步流程
  - PDF章节：第9章 - 交易准备
  - 5个步骤：
    1. 趋势过滤
    2. 位置过滤
    3. 成交量确认
    4. 波动率检查
    5. 等待入场信号

## 阶段三：风险管理流程（第12-15周）
**教材**：04 进阶篇 YTC卷四：您的交易业务.pdf

### 知识点清单

#### 3.1 仓位管理
- **知识点18**：2%规则
  - PDF章节：第3章 - 资金管理
  - 计算公式：
    ```
    仓位 = (总资金 × 2%) ÷ 止损距离
    ```

- **知识点19**：凯利公式
  - PDF章节：第3章 - 仓位规模
  - 公式：
    ```
    f = (bp - q) / b
    ```
    - b = 盈亏比
    - p = 胜率
    - q = 1-p

#### 3.2 止损策略
- **知识点20**：结构性止损
  - PDF章节：第4章 - 止损设置
  - 设置位置：
    - 支撑位下方2-3%
    - 前期低点下方
    - 上升趋势线下方

- **知识点21**：波动率止损
  - PDF章节：第4章 - ATR止损
  - 计算方法：
    ```
    止损位 = 入场价 ± (ATR × 1.5或2)
    ```

- **知识点22**：时间止损
  - PDF章节：第4章 - 时间管理
  - 规则：
    - 持有超过10天未达到预期
    - 趋势减弱时退出

#### 3.3 盈亏比管理
- **知识点23**：盈亏比计算
  - PDF章节：第5章 - 盈亏比
  - 计算公式：
    ```
    盈亏比 = (目标价 - 入场价) / (入场价 - 止损价)
    ```

- **知识点24**：分批止盈
  - PDF章节：第5章 - 出场策略
  - 分配方案：
    - 50%仓位在1R（风险×1）
    - 30%仓位在2R
    - 20%仓位在3R+

- **知识点25**：追踪止损
  - PDF章节：第5章 - 追踪止损
  - 启动时机：
    - 突破1R后：移至保本
    - 突破2R后：移至+1R
    - 突破3R后：移至+2R

#### 3.4 组合管理
- **知识点26**：相关性管理
  - PDF章节：第6章 - 投资组合
  - 关键要点：
    - 避免过度集中
    - 不同板块分散
    - 最大持仓5只股票

- **知识点27**：回撤控制
  - PDF章节：第6章 - 风险控制
  - 控制规则：
    - 单笔最大亏损2%
    - 日最大亏损6%
    - 月最大亏损15%
