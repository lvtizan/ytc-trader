## 设计特点

### 蓝紫色科技光晕背景
所有页面现在包含多层科技感光晕效果：

#### 1. 主光晕层 (body::after)
- **顶部大光晕**: 蓝紫色渐变光晕
- **位置**: 页面顶部和中心区域
- **颜色**:
  - rgba(99, 102, 241, 0.15) - 靛蓝色
  - rgba(6, 182, 212, 0.12) - 青色
  - rgba(139, 92, 246, 0.1) - 紫色
  - rgba(59, 130, 246, 0.08) - 蓝色
- **动画**: 8秒呼吸效果

#### 2. 侧边光晕层 (.tech-glow)
**左侧光晕**:
- 颜色: 靛蓝 → 紫色渐变
- 动画: 12秒左右移动

**右侧光晕**:
- 颜色: 青色 → 蓝色渐变
- 动画: 10秒左右移动

### 配色方案

```css
/* 蓝紫色系 */
--primary: #6366f1        /* 靛蓝 */
--primary-light: #818cf8  /* 浅靛蓝 */
--primary-dark: #4f46e5   /* 深靛蓝 */

/* 强调色 */
--accent: #06b6d4         /* 青色 */
--accent-light: #22d3ee   /* 浅青色 */

/* 紫色辅助 */
--purple: #8b5cf6         /* 紫色 */
--blue: #3b82f6           /* 蓝色 */
```

## 实现方法

### HTML结构
所有页面需要在 `<body>` 标签后立即添加：

```html
<body>
    <!-- 科技光晕背景层 -->
    <div class="tech-glow"></div>

    <!-- 页面内容 -->
    <div class="app-container">
        ...
    </div>
</body>
```

## 维护说明

### 修改颜色
如需调整光晕颜色，修改 `modern-premium.css` 中的：
- `body::after` - 主光晕
- `.tech-glow::before` - 左侧光晕
- `.tech-glow::after` - 右侧光晕

### 调整动画速度
修改 `animation` 属性中的秒数：
- `glow-pulse 8s` - 主光晕呼吸
- `side-glow-left 12s` - 左侧移动
- `side-glow-right 10s` - 右侧移动

### 调整强度
修改 `rgba()` 中的透明度值：
- 当前范围: 0.03 - 0.15
- 更强效果: 增加到 0.1 - 0.25
- 更弱效果: 减少到 0.02 - 0.1
