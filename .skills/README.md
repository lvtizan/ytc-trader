## 添加新 Skills

当发现重复性工作时，创建新的 Skill：

1. **识别重复模式**
   - 哪些步骤每次都重复？
   - 哪些规则容易忘记？

2. **文档化流程**
   - 写下详细步骤
   - 包含代码模板
   - 说明注意事项

3. **测试验证**
   - 按照 Skill 执行
   - 确保结果正确
   - 优化流程

4. **保存到 .skills 目录**
   - 使用清晰的文件名
   - 添加描述和使用场景
