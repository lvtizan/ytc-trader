# YTC Trader - 快速启动指南

## 🚀 立即开始使用

### 方法一：浏览器直接打开（最简单）

1. 打开Finder，进入项目目录
2. 双击 `index.html` 文件
3. 系统会在默认浏览器中打开

---

### 方法二：使用本地服务器（推荐）

#### 已启动服务器

服务器已在后台运行：
```
http://localhost:8888
```

直接在浏览器中访问上述地址即可！

#### 手动启动服务器

如果服务器已停止，可以使用以下命令重新启动：

```bash
cd ytc-trader
python3 -m http.server 8888
```

然后在浏览器中访问：http://localhost:8888

- 导出进度备份

## 📊 功能说明

### 自动保存
所有进度自动保存到浏览器，无需手动保存

### 导出进度
在"学习进度"页面点击"导出进度"按钮，可导出JSON格式的进度文件

### 搜索功能
使用顶部搜索图标或导航中的搜索功能，快速查找内容

## 💡 使用技巧

1. **添加书签**：将主页添加到浏览器书签，方便快速访问
2. **快捷键**：使用浏览器快捷键（Cmd/Ctrl + D）添加书签
3. **多标签页**：在不同标签页中打开不同阶段和工具，便于参考

---

## 🐛 遇到问题？

1. **样式异常**：清除浏览器缓存后刷新
2. **进度丢失**：检查是否使用了隐身模式（隐身模式不会保存数据）
3. **功能异常**：尝试使用Chrome或Firefox浏览器
